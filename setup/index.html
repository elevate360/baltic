      # Initial Theme Setup

      # Installation

      You can install the Baltic theme in many different ways. We have listed some of them.

      # Automatic Installation

      1. Log in to your WordPress dashboard and go to Appearance > Themes.
      2. Click the Add New button at the top of the page.

      1. Type Baltic on theme search form.
      2. After the theme reveal on the page, click Install. After the installation finished, click Activate.

      # Manual Installation

      1. Log in to your WordPress dashboard and go to Appearance > Themes.
      2. Click the Add New button at the top of the page.

      1. Click the Upload Theme button at the top of the new page.
      2. Click the Choose File button and find the file for your theme on your computer. The file will typically be named with the name of the theme and have an ending of .zip (example: baltic.1.1.1.zip or atlantic.1.1.0.zip).
      3. Click the Install Now button once you have selected the theme file for upload.

      1. After installing the Baltic theme you will be redirected to the new screen. Click the Activate link to activate the theme.

      # WP-CLI Installation

      Under your root directory of WordPress installation theme, you can type:

      wp theme install baltic

      To activate the theme, you can type this command:

      wp theme activate baltic

      # Plugins recommendation

      After the theme activated there will be a notification. In order to get all features that Baltic theme can give. It is recommended to install and activate these plugins below:

      1. Baltic Kit - This plugin is a core features of Baltic theme can offer.
      2. WooCommerce - WooCommerce is a free eCommerce plugin that allows you to sell anything, beautifully. Built to integrate seamlessly with WordPress, WooCommerce is the world’s favorite eCommerce solution that gives both store owners and developers complete control.
      3. YITH WooCommerce Wishlist - Offer to your visitors a chance to add the products of your woocommerce store to a wishlist page..
      4. Elementor - A page builder that delivers high-end page designs and advanced capabilities, never before seen on WordPress.
      5. Contact Form 7 - Contact Form 7 can manage multiple contact forms, plus you can customize the form and the mail contents flexibly with simple markup. The form supports Ajax-powered submitting, CAPTCHA, Akismet spam filtering and so on.
      6. WP Term Images - This plugin allows users to assign images to any visible category, tag, or taxonomy term using the media library, providing a customized look for their taxonomies.
      7. WP Instagram widget - WP Instagram widget is a no fuss WordPress widget to showcase your latest Instagram pics. It does not require you to provide your login details or sign in via oAuth.
      8. One Click Demo Import - This plugin enable you to install pre-defined demo content in one click.

      # Recommended plugins installation

      1. Click the Begin activating plugins.

      1. At the top left of the table, change the Bulk Actions into Install.
      2. Select all plugins by checking the checkbox at the top left.
      3. Click Apply to start the installation.

        WARNING

        The installation progress might take some time depend on your server.

      1. After the installation finished, click the Return to Required Plugins installer.

      # Recommended plugins activation

      The plugins activation steps are very similar to the installation.

      1. At the top left of the table, change the Bulk Actions into Activate.
      2. Select all plugins by checking the checkbox at the top left.
      3. Click Apply to start the activation.

      1. After the activation finished, click Return to the Dashboard link.

      # Demo Import

      The Baltic theme has pre-made designs and ready to import for your site.

      Important

      The demo import screen only available if you install and activate the Once Click Demo Import plugin.

      1. Navigation to your WordPress dashboard and go to Appearance > Import Demo data.
      2. Select any demo that suits your need and then click the Import button.

      1. Click Yes, import! to confirm the import progress.

      WARNING

      The import progress might take some time depend on your server.

      The green notification will appear if the demo successfully imported.
